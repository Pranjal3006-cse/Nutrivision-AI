# 🍎 Nutrivision AI – Intelligent Food Analysis Agent

## 🚀 Overview
Nutrivision AI is an AI-powered agentic system that analyzes food inputs (image or text) and provides detailed nutritional insights along with personalized health recommendations.

This project demonstrates autonomous decision-making by integrating AI models, data retrieval, and intelligent response generation.

---

## 🧠 Agentic Workflow

1. User provides input (food image or name)
2. AI model identifies the food item
3. System retrieves nutritional data (calories, macros, etc.)
4. AI analyzes the data
5. Agent generates health recommendations

➡️ This end-to-end pipeline showcases true **agentic behavior** (input → reasoning → action)

---

## ⚙️ Features

- 🍽 Food Recognition (Image/Text)
- 🔍 Nutritional Analysis (Calories, Protein, Carbs, Fat)
- 📊 Intelligent Health Suggestions
- ⚡ Fast API-based backend
- 🤖 AI-driven decision making
